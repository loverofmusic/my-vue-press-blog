### 常用快捷键一览表：

| 功能                                |   快捷键                             | 
| ---------------------------------  | ----------------------------------- |
| 显示隐藏文件                         | ⌘⇧.  (Command + Shift + .)          |
| 区域截图（默认保存至桌面）              | ⌘⇧.  (Command + Shift + 4)          |
| 全屏截图（默认保存至桌面）              | ⌘⇧.  (Command + Shift + 3)          |
| 打开开发者工具              | Command + Option + i         |
